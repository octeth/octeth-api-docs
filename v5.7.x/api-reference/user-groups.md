# User Groups

The User Groups API allows you to manage user groups within the Octeth platform, providing functionality to create, update, retrieve, and organize users into different permission levels and access groups.

::: warning Note
This section is under development. Documentation will be available soon.
:::

## API Endpoints Covered

This section will document the following user group management endpoints:

- **User Group Management**
  - `UserGroup.Create` - Create a new user group
  - `UserGroup.Update` - Update existing user group details
  - `UserGroup.Delete` - Delete a user group
  - `UserGroup.Get` - Retrieve specific user group information

- **User Group Listing**
  - `UserGroups.Get` - Retrieve list of all user groups
  - `UserGroups.GetHierarchy` - Get user group hierarchy structure

- **User Group Assignment**
  - `UserGroup.AddUser` - Add users to a group
  - `UserGroup.RemoveUser` - Remove users from a group
  - `UserGroup.GetUsers` - Get users in a specific group

- **Permission Management**
  - Group permission settings and access control
  - Feature limitations and quotas per group
  - Default settings and configurations
  - Group-specific email sending limits
