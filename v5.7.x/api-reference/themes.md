# Themes

The Themes API provides comprehensive theme management functionality for customizing the appearance and branding of your email templates, forms, and user interfaces within the Octeth platform.

::: warning Note
This section is under development. Documentation will be available soon.
:::

## API Endpoints Covered

This section will document the following theme management endpoints:

- **Theme Management**
  - `Theme.Create` - Create a new custom theme
  - `Theme.Update` - Update existing theme settings
  - `Theme.Delete` - Remove a custom theme
  - `Theme.Get` - Retrieve specific theme details

- **Theme Listing**
  - `Themes.Get` - List all available themes
  - `Themes.GetBuiltin` - Retrieve built-in system themes
  - `Themes.GetCustom` - List user-created custom themes
  - `Themes.GetActive` - Get currently active theme

- **Theme Application**
  - `Theme.Apply` - Apply theme to templates or forms
  - `Theme.SetDefault` - Set default theme for account
  - `Theme.Preview` - Generate theme preview
  - `Theme.Reset` - Reset to default theme settings

- **Theme Features**
  - Custom color schemes and branding
  - Typography and font management
  - Layout and spacing customization
  - Logo and image integration
  - CSS customization and overrides
  - Responsive design themes
  - Dark mode and theme variations
  - Theme inheritance and cascading
  - Brand consistency enforcement
  - Multi-brand theme support
