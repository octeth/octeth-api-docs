# Subscriber Activities

The Subscriber Activities API provides comprehensive tracking and monitoring of subscriber interactions, behaviors, and engagement metrics within the Octeth platform.

::: warning Note
This section is under development. Documentation will be available soon.
:::

## API Endpoints Covered

This section will document the following subscriber activity tracking endpoints:

- **Activity Tracking**
  - `Subscriber.Get.Activity` - Retrieve subscriber activity history
  - `Subscriber.TrackActivity` - Record custom subscriber activities
  - `Subscriber.GetEngagement` - Get subscriber engagement metrics
  - `Subscriber.GetInteractions` - Track subscriber interactions

- **Administrative Activity Monitoring**
  - `Admin.Users.Activity` - Monitor user activities (admin only)
  - `Admin.Subscribers.Activity` - Track subscriber activities across accounts
  - `Admin.GetActivityLogs` - Retrieve system-wide activity logs

- **Activity Types**
  - Email opens and click tracking
  - Website visit tracking
  - Form submission activities
  - Purchase and conversion events
  - Subscription status changes
  - Custom event tracking

- **Analytics and Reporting**
  - Activity aggregation and statistics
  - Behavior pattern analysis
  - Engagement scoring and metrics
  - Activity timeline and history
  - Real-time activity monitoring
